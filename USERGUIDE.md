# 使用示例

## 基础使用流程

### 1. 打开视频文件
- 启动应用后，点击界面右上角的"打开视频"按钮
- 在文件选择对话框中，选择要编辑的视频文件
- 支持的格式：MP4, AVI, MOV, MKV, WMV, FLV, WebM

### 2. 预览视频
- 视频加载后，使用内置播放器预览
- 可以播放、暂停、拖动进度条
- 点击时间线可以快速跳转到指定位置

### 3. 选择裁剪范围
#### 方法一：手动输入时间
- 在"开始时间"输入框中输入开始时间（格式：HH:MM:SS）
- 在"结束时间"输入框中输入结束时间

#### 方法二：使用当前位置按钮
1. 播放视频到想要开始的位置
2. 点击"设为当前位置"按钮（开始时间旁边）
3. 继续播放到想要结束的位置
4. 点击"设为当前位置"按钮（结束时间旁边）

### 4. 保存视频
#### 保存视频
- 点击"保存视频"按钮
- 在原文件同目录下生成 `原文件名_edited.mp4`
- 例如：原文件为 `video.mp4`，保存后为 `video_edited.mp4`

#### 另存为
- 点击"另存为..."按钮
- 在原文件同目录下生成带时间戳的新文件
- 例如：`video_20260723_143025_edited.mp4`

## 实际案例

### 案例1：裁剪视频片段
```
原始视频: 10分钟
目标: 提取第2分钟到第5分钟的片段

步骤:
1. 打开视频文件
2. 播放到第2分钟位置
3. 点击开始时间的"设为当前位置" → 00:02:00
4. 播放到第5分钟位置  
5. 点击结束时间的"设为当前位置" → 00:05:00
6. 选择时长显示为 00:03:00
7. 点击"保存视频"
```

### 案例2：删除开头和结尾
```
原始视频: 15分钟
目标: 删除开头1分钟和结尾2分钟，保留中间12分钟

步骤:
1. 打开视频文件
2. 在开始时间输入框输入: 00:01:00
3. 在结束时间输入框输入: 00:13:00
4. 确认选择时长为 00:12:00
5. 点击"另存为..." 生成新文件
```

### 案例3：快速预览后裁剪
```
场景: 观看视频时发现某个片段很精彩，想单独保存

步骤:
1. 打开视频，正常播放观看
2. 发现精彩片段开始时，点击"设为当前位置"（开始时间）
3. 精彩片段结束时，点击"设为当前位置"（结束时间）
4. 点击"保存视频"
```

## 文件管理器集成使用

### Windows
```
场景: 在文件夹中快速编辑视频

步骤:
1. 在文件管理器中找到视频文件
2. 右键点击视频文件
3. 选择"打开方式" → "视频粗剪工具"
4. 应用自动打开并加载该视频
5. 编辑后保存
```

### macOS
```
场景: 设置默认使用此工具打开视频

步骤:
1. 右键点击任意视频文件
2. 选择"显示简介"
3. 在"打开方式"中选择"视频粗剪工具"
4. 点击"全部更改"
5. 以后双击视频文件会自动用此工具打开
```

## 键盘快捷键（应用内）

- `空格键`: 播放/暂停
- `左箭头`: 快退5秒
- `右箭头`: 快进5秒
- `上下箭头`: 音量调节

## 提示和技巧

1. **精确裁剪**: 在时间输入框直接输入精确时间，格式为 HH:MM:SS
2. **预览选择**: 点击时间线上的选择区域（半透明白色区域）可以预览选择的片段
3. **批量处理**: 可以连续打开多个视频文件进行编辑
4. **原文件保护**: "保存视频"不会覆盖原文件，而是生成新的 _edited 文件

## 性能建议

- 短片段（<5分钟）：处理速度快，适合快速预览和保存
- 中等片段（5-30分钟）：处理需要几分钟，建议在后台处理
- 长片段（>30分钟）：处理时间较长，建议在空闲时处理

## 输出质量说明

应用使用 H.264 视频编码和 AAC 音频编码：
- 视频比特率：自动（保持原视频质量）
- 音频比特率：192 kbps
- 编码速度：中等（balance 速度和质量）

如需更高或更低的输出质量，可以修改代码中的 FFmpeg 参数。
